------------------
LISEZMOI de Verbot 5
------------------

Verbot® 5 est livré avec deux applications. Verbot® 5 Player et le script Verbot
Interface (ou éditeur).

Verbot® 5
Verbot® charge des bases de connaissances et affiche un personnage animé pouvant
être interagi avec une conversation textuelle. Contenu dans un
Knowledgebase Rule permet au personnage Verbot® d’ouvrir des applications de bureau,
ainsi que d'effectuer diverses animations.

Interface de script Verbot
Verbot Scripting Interface est utilisé pour créer des bases de connaissances personnalisées.
L’interface facile à utiliser vous permet de créer des règles contenant des entrées et des
Les sorties. Les entrées peuvent utiliser les fichiers Verbot® Synonym pour transformer le script en script.
brise. Les sorties peuvent contenir des balises spéciales pouvant ouvrir et exécuter des applications,
ainsi que d'animer le Verbot® de plusieurs façons.

Avec la technologie Verbot® 5, la création de votre propre agent intelligent devient aussi simple que
simple comme taper une phrase ou une commande dans un texte en langage naturel.
Le manuel d’aide en ligne contient des informations qui vous permettront de maîtriser rapidement
Créez rapidement votre propre personnalité Verbot®. Bien sûr vous n'avez pas à
créez votre propre base de connaissances Verbot®, mais le faire est à moitié amusant!
Aide en ligne:
http://www.verbot.com/online_help/

-----------------------------------
Instructions d'installation pas à pas:
-----------------------------------

1. Si vous ne possédez pas Microsoft .NET Framework version 2.0 ou supérieure
installé, allez à http://www.windowsupdate.com pour l’installer.

2. Téléchargez et installez le moteur L & H British English TTS à partir de:
http://www.microsoft.com/msagent/downloads/user.asp#tts

3. Si vous utilisez le système d'exploitation Windows XP, téléchargez le fichier SAPI 4.0 à l'adresse:
http://www.microsoft.com/msagent/downloads/user.asp#sapi

4. Copiez les fichiers de l'agent ka2004.acs et sylvie2004.acs dans le répertoire suivant:
Sous Windows XP:
C: \ Windows \ msagent \ chars

Sous Windows 2000:
C: \ WINNT \ msagent \ chars

5. Vous pouvez maintenant exécuter le programme Verbot4Engine.exe pour discuter avec Sylvie®.
Remarque: au démarrage de Verbot 4.0, l’agent par défaut est l’agent Merlin,
pour le changer, utilisez le menu Outils-> Options, cliquez sur l'onglet Personnaliser, choisissez
le fichier sylvie2004.acs dans la liste déroulante Fichier de l'agent.

6. Si vous souhaitez modifier la KnowledgBase ou créer une nouvelle base de connaissances, exécutez
le programme Verbot4Editor.exe.
