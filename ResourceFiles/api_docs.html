DevBot4Engine

Classe de réponse

Présentation:
L'objet de réponse représente les résultats d'un appel à DevBot4Engine.GetReply.
Propriétés:
AgentText (string) - Texte à prononcer (utilisé avec MS Agent)
Cmd (string) - Texte de commande pour la sortie
KBItem (KnowledgeBaseItem) - La base de connaissances contenant la règle de correspondance
RuleId (string) - Identifiant de la règle qui a été déclenchée
Texte (chaîne) - Texte de sortie à afficher

Méthodes:(rien de spécial, juste ceux hérités de Object)


Classe d'état Présentation:
Une des deux entrées principales de DevBot4Engine.GetReply. Représente l'état actuel du moteur, notamment: la dernière règle à déclencher, les valeurs des variables en cours et les bases de connaissances actives.
Propriétés:
CurrentKBs (ArrayList) - ArrayList des chaînes qui sont les chemins d'accès aux fichiers de la base de connaissances active.
Lastfired (string) - Identifiant de la dernière règle à déclencher.
Lastinput (string) - Texte saisi précédemment par l'utilisateur.
LastRefreshedTime (DateTime) - Dernière fois que l'objet d'état a été actualisé
Vars (Hashtable) - Hashtable représentant les variables actuelles du moteur.
Méthodes:
void LoadVars (string filepath) - Charge un ensemble de variables à partir du fichier donné.
void SaveVars (string filepath) - Enregistre le fichier State.Vars actuel dans le fichier indiqué.

Classe DevBot4Engine Présentation:
Le moteur DevBot de base. Est responsable de trouver des correspondances avec une entrée utilisateur donnée et d'activer la règle appropriée.
Propriétés:
(aucun)
Méthodes:
CompiledKnowledgeBase AddCompiledKnowledgeBase (string stPath) - Charge une base de connaissances compilée à partir du chemin de fichier indiqué.
Reply GetReply (chaîne de caractères, State state): recherche la meilleure correspondance dans la base de connaissances disponible et déclenche la règle correspondante. Le paramètre "input" est le texte d'entrée de l'utilisateur et le paramètre "state" est l'état actuel de l'utilisateur.
